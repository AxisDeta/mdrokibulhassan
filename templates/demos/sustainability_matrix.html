Research Methodology

1

Data Description

Utilizing a 35-variable dataset from a Fashion & Beauty startup. Features include Panel, Purchase, Dev/Proto, Gender, Category, and Supplier (A-G), comprising ~100 elements.

2

Data Preprocessing

Strategic noise handling and missing value imputation. Dataset split into 80% training and 20% testing. Categorical features processed via Label Encoding followed by One-Hot Encoding.

3

Model Matrix

A comparative analysis of four supervised classifiers: KNN (Manhattan distance), Naïve Bayes (Probabilistic), Random Forest (200 trees), and Neural Networks (Perceptron).
